Privacy Policy

Last updated: 05/26/2025

This MyPassSafe application is developed with privacy as a top priority. Below is an explanation of how the app handles your data.

1. Data Collection

The app does not collect, transmit, or store any personal data. All actions and data remain strictly on your device.

2. Data Storage

  • All saved passwords and the master password are stored locally on the device.
  • Data is stored in an encrypted SQLite database using cryptographic algorithms via the crypto-js library.
  • The app does not connect to the internet, use cloud storage, or interact with any external servers.

3. Security

  • Data security depends on the strength and confidentiality of your master password. The developer has no access to the master password and cannot help recover it.
  • The user is responsible for securely handling any exported data files.

4. Third-Party Libraries

The app only uses open-source libraries that are included locally:

  • sql-wasm.sql, sql-wasm.wasm — for SQLite support.
  • crypto-js.min.js — for data encryption.

No data is ever transmitted online.
